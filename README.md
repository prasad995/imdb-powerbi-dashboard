# IMDb Movies Power BI Dashboard

## Dashboard Preview
![Dashboard](dashboard%20Screenshot.png)

## About
An interactive Power BI dashboard analyzing Top IMDb Movies dataset sourced from Kaggle. The dashboard provides insights into movie ratings, release trends, genres, and directors through dynamic visualizations and multi-field slicers.

## Dataset
- **Source:** Kaggle
- **Content:** Top IMDb rated movies
- **Fields:** Title, Stars, Director, Category, Censor Board Rating, Release Year, IMDb Rating

## Key Insights
- Average IMDb rating across top movies: **7.96**
- Movie production increased significantly post **2000**
- Most common categories include **Action, Adventure, Drama**
- Filtered by **11 Censor Board ratings** including U/A, PG-13, R, Unrated

## Tools Used
| Tool | Purpose |
|------|---------|
| Power BI | Dashboard creation & visualization |
| Power Query | Data cleaning & transformation |
| DAX | Calculated measures & aggregations |
| Kaggle | Dataset source |

## Features
- Multi-field slicers for Title, Director, Category, Stars, Censor Board Rating
- Release year trend analysis (1900–2020)
- Average IMDb rating visualization
- Interactive cross-filtering across all visuals
